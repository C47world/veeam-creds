PS скрипт для декрипта veeam
```
1. заходим на сервер с виамом
2. скачиваем скрипт https://github.com/C47world/veeam-creds/blob/main/Veeam-Get-Creds.ps1
3. открываем elevated powershell (run as administrator)
4. проваливаемся в папку со скриптом и выполняем .\Veeam-Get-Creds.ps1
на выходе получаем креды в виде логин пас
```
